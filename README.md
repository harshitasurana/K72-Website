# 🧠 Problem Statement  

Modern websites often lack engaging animations and smooth user interactions, leading to:

❌ Poor user engagement  
❌ Static and less interactive UI  
❌ Lack of smooth transitions  
❌ Non-responsive design across devices  

---

# 💡 Solution  

This project provides a **modern animated frontend website** that:

✔ Uses GSAP for smooth animations  
✔ Delivers highly interactive UI/UX  
✔ Ensures full responsiveness across devices  
✔ Enhances user engagement with motion effects  

---

# 🏗️ System Architecture  

Frontend (React.js) → GSAP Animation Layer → UI Rendering (Vite Build Tool)

# ⚙️ Installation  

## 1️⃣ Clone Repository  

```bash
git clone https://github.com/harshitasurana/K72-Website.git
cd K72-Website
```

## 2️⃣ Install Dependencies

```
npm install
```

## 3️⃣ Run Application
```
npm run dev

```

## 👉 Open in browser:

http://localhost:5173
